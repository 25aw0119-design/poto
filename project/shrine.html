謎先の神社アプリ

企画 / Figma / Illustrator / Photoshop /
HTML.CSS / JavaScript

概要

神社で発見した課題に対して、謎先の神社｜探索型謎解きアプリというデザインしました。

目的

境内でなぞなぞを​することでリラックスできる。

ターゲット

悩み、​ストレスを​抱えてる​20代後半〜30代​

デザイン

考える謎解きではなく、探すゲーム感覚の体験で神社の中を巡り、手がかりを探す遊び方で「考える」謎々より「見つける」楽しさを重視した上で、神社の情報につなぎますので、自分が好きな神社を選ばられ、ハートにつけて、予約にも追加ですることができます。

制作期間

３週間
